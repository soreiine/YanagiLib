# YanagiLib
自分用に作成したPaperプラグインの作成を補助するユーティリティライブラリ

## 使い方
### Maven
```xml
<project>
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>

    <dependencies>
        <dependency>
            <groupId>com.github.soreiine</groupId>
            <artifactId>YanagiLib</artifactId>
            <version>0.1</version>
        </dependency>
    </dependencies>
</project>
```
